[![npm](https://img.shields.io/npm/dt/anc.svg)](https://www.npmjs.com/package/anc)

A Node.js module boilerplate made simple

## Installation and Usage

Prerequisites: [Node.js](https://nodejs.org/en/) (>=8.11.1), npm version 3+.

### Global Installation

#### yarn

```
yarn global add anc
```

#### npm

```
npm i -g anc
```

## Usage

```
anc init
```
